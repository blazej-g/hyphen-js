HyphenJs, an Angular module made to simplify data model creation with ability to work in offline mode

HyphenJs is made to support build process of nontrivial Angular applications, where reliable and clean data model is required. HyphenJs minimize time and effort required to be spent on application data layer. Developers can focus on creating rich users interfaces instead of data modeling and restful api calls implementation.
